## 🚀 **Quick Start Guide**

### **1. Install Dependencies**
```bash
cd MasChat-main
quick-install.bat
```

### **2. Set Up Environment**
```bash
copy .env.template .env
# Edit .env with your values
```

### **3. Deploy Contracts**
```bash
npm run deploy:mumbai
```

### **4. Start the App**
```bash
npm start
```
